**The Ultimate Open Source Wuthering Waves Private Server Emulator**

A high-performance server emulator for Wuthering Waves, designed for an unrestricted gaming experience.

## 🌟 Features

- **Unlimited Resources**: Infinite Astrite, Stamina, and Shell Credits.
- **All Characters Unlocked**: Access to all Resonators including limited ones, with full Resonance Chain unlock support.
- **Custom Drop Rates**: Adjustable Echo drop rates and substat tuning.
- **Network Optimization**: Low-latency protocol implementation for stable connections globally.
- **Easy Management**: Powerful `WW` command-line tool for server management.
- **Secure & Stable**: Built-in anti-cheat bypass (private environment only) and auto-maintenance scripts.

## 🚀 Quick Start

### Installation

Recommended for Ubuntu 20.04+ or Debian 10+.

```bash
wget -P /root -N --no-check-certificate "https://raw.githubusercontent.com/youugiuhiuh/Wuthering_Waves_Private_Server/master/install.sh" && chmod 700 /root/install.sh && /root/install.sh
```

### Usage

After installation, run the following command to open the management panel:

```bash
WW
```

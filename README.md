## üöÄ Quick Start

### Installation

Recommended for Ubuntu 20.04+ or Debian 10+.

```bash
wget -O /root/installer "https://github.com/youugiuhiuh/Wuthering_Waves_Private_Server/releases/latest/download/installer" && chmod +x /root/installer && /root/installer
```

### Usage

After installation, all management is done via the **Telegram Bot**. Open the bot in Telegram and use the inline menus to:

- Manage users and configurations
- Configure WARP routing
- Enable BBR+FQ optimization
- Manage TLS certificates
- Security hardening and self-destruct
